KEY FILES EXPLAINED
===================

PUBLIC DIRECTORY
----------------
index.html      - Main application interface
                  Entry point for users
                  Contains quiz UI structure

css/style.css   - All styling and design
                  Responsive layout
                  Theme and colors

js/app.js       - Quiz logic and functionality
                  Timer, navigation, scoring
                  Question randomization
                  Results calculation

data/quiz_data.json - Quiz content
                      30 sailing theory questions
                      Configurable settings
                      (time limit, passing score)


DOCKER FILES
------------
Dockerfile          - Creates nginx-based container
                      Copies public/ files
                      Exposes port 80

docker-compose.yml  - Easy one-command deployment
                      Maps port 8080 to container
                      Defines network settings

.dockerignore       - Excludes unnecessary files
                      from Docker image
                      (docs, scripts, etc.)


SCRIPTS DIRECTORY
-----------------
pdf_to_json.py      - Converts PDF questions to JSON
                      Useful for adding new questions
                      Requires pdfminer.six

requirements.txt    - Python dependencies for scripts

DEPLOYMENT OPTIONS
==================

1. DOCKER (Recommended)
   Command: docker-compose up -d
   Access:  http://localhost:8080
   Benefits: Zero local dependencies
            Production-ready
            Easy to deploy

2. LOCAL DEVELOPMENT
   Command: python -m http.server 8000
   Path:    cd public/ first
   Access:  http://localhost:8000
   Benefits: Quick for development
            No Docker needed
